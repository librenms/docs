We have available for download a pre-built image based on Ubuntu 16.04 LTS. Details of the image are below:

The image is built with VirtualBox, a vmdk is provided along with an ova which was exported using OFV 1.0 version. These should be supported in VMWare Fusion, Workstation, Player and VirtualBox.

  • NOTE: It's highly recommended that you update by doing the following. 
cd /opt/librenms
./daily.sh
  • Any issues and or help with these images should be reported via Community Fourm or our Discord server

Setup

  • UK Keyboard
  • Etc/UTC Timezone
  • LVM for disk setup
  • 4 Poller Wrapper threads

Software

  • PHP 7
  • MariaDB
  • Syslog-ng

Features

  • Oxidized install but not configured
  • Weathermap plugin enabled
  • Billing enabled
  • RRDCached enabled
  • Service checks enabled
  • Syslog enabled
